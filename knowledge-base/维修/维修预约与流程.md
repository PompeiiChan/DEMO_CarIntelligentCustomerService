# 维修预约与流程

## 预约入口

- 优先引导用户通过极氪 App、官方客服或当地极氪授权服务中心预约。
- 收集最小必要字段：城市、车型、问题类型、是否可继续行驶、联系方式；售后高风险补充 VIN/当前位置/安全状态。

## 到店流程

- 到店接待 -> 故障描述确认 -> 车辆检测 -> 报价/质保判断 -> 用户确认 -> 维修 -> 交车验收。
- 涉及费用争议、质保争议、事故维修、保险理赔，客服应创建人工工单。

## 报价规则

- Demo 仅提供参考性保养费用；维修价格以服务中心检测和报价为准。
