# 不可承诺事项

> 这份清单用于约束自动客服、RAG 回复和人工辅助话术。命中后不要为了显得“有帮助”而给确定性结论。

| 不可承诺事项 | 原因 | 正确动作 | 可说范围 |
| --- | --- | --- | --- |
| 最终成交价/库存/交付周期 | 价格和库存强动态，门店差异大 | 以极氪 App/官网/当地门店实时确认为准 | 可以说明本地资料中的历史价格和更新时间。 |
| 金融审批结果/月供/利率 | 受资质、征信、金融机构政策影响 | 转人工或金融顾问确认 | 可以说明常见材料和审批流程。 |
| 退款/退定/改配一定成功 | 涉及合同、支付和排产状态 | 人工核查订单 | 可以收集订单号和争议点。 |
| 事故责任判断 | 属于交警/保险/司法判断 | 提示报警、保险报案、救援 | 可以给事故现场安全步骤。 |
| 车辆是否还能继续行驶 | 高风险，远程无法判断 | 安全停车并联系救援/服务中心 | 尤其红灯、制动、转向、高压故障。 |
| 高压系统拆修指导 | 存在触电和车辆损坏风险 | 到极氪授权服务中心处理 | 只能做安全提示。 |
| 质保一定通过/免费更换 | 需检测、保养记录、车主身份和条款判断 | 服务中心/官方质保审核 | 可以说明质保条件和排除项。 |
| 召回范围泛化 | 召回有版本、生产日期、VIN 范围 | 官方召回平台/服务中心核验 VIN | 001 WE 召回不可泛化为所有 001。 |
| 地方补贴一定可领/可叠加 | 地方政策和资格审核差异大 | 当地平台/门店确认 | 可以说明国家政策框架。 |
| 智能驾驶等于自动驾驶 | 功能边界和法律责任风险 | 统一称智能驾驶辅助 | 强调驾驶员始终负责接管。 |
| 医疗/法律/保险理赔最终建议 | 超出客服系统职责 | 专业机构/人工专员处理 | 可以提供流程性引导。 |

## 通用兜底句

- “这个信息需要以极氪 App、官网、购车协议、用户手册、服务中心或当地门店实时确认为准。”
- “我可以先帮您整理已知信息和需要核验的字段，再交给人工客服继续处理。”
- “涉及安全风险时，建议您先停止操作并联系官方救援/服务中心。”
