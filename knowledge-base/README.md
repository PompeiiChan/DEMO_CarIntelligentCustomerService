# 汽车智能客服知识库

这是从现有 `docs/`、`data/zeekr_vehicles.sqlite` 和 2026-06-11 外部复核信息迁移后的客服知识库。

## 使用方式

- 售前车型、配置、对比：从 `车型/` 开始。
- 价格、权益、金融、置换：从 `价格/` 开始，并优先查看时效提醒。
- 保养、充电、日常用车：从 `保养/` 开始。
- 故障、维修、救援：从 `维修/` 开始，高风险场景直接转人工/救援。
- 质保、OTA、召回、服务承诺：从 `政策/` 开始。
- 用户原话、意图和应检索文件：从 `问法库/` 开始。
- 转人工规则、工单摘要和安抚话术：从 `人工介入/` 开始。
- 不可承诺事项和兜底边界：从 `安全边界/` 开始。

## 重要边界

- 本知识库支持客服回答，不替代官方 App、官网、购车协议、用户手册和服务中心判断。
- 任何过期、未知、第三方来源或门店口径的信息，都只能作为线索，不能作为客服承诺。
- 迁移清单见 `manifest.md`；来源与时效规则见 `来源与时效.md`。
