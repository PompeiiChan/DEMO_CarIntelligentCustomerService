# 极氪 App 使用指南

> 极氪 App（原名极氪生活）是车主管理车辆、预约服务、接收 OTA 推送的主要工具。功能以 App 当前版本为准，以下为通用功能说明。

## 下载与注册

- 在 App Store 或应用宝/华为应用市场搜索「极氪」下载官方 App。
- 使用手机号注册，完成实名认证后可绑定车辆。
- 同一账号支持绑定多辆车，可在 App 内切换。

## 绑定车辆

首次使用需绑定车辆，步骤：

1. 打开极氪 App，进入「我的」→「车辆管理」。
2. 点击「添加车辆」，输入 VIN 码（车架号）或车牌号。
3. 系统验证后完成绑定，需确保车辆已在极氪系统中登记（提车时销售会协助完成）。
4. 绑定后，App 首页显示车辆状态、电量、位置等信息。

如绑定失败，联系门店销售或极氪客服确认 VIN 码是否已正确录入系统。

## 远程控制功能

绑定车辆后，App 支持以下远程操作：

| 功能 | 说明 |
| --- | --- |
| 远程解锁/上锁 | 通过 App 控制车门，需车辆在网络覆盖区域 |
| 远程开启空调 | 提前预热或预冷车厢，节省等待时间 |
| 远程寻车 | 显示车辆实时位置，支持多角度全景照片寻车 |
| 远程鸣笛/闪灯 | 在停车场找车时使用 |
| 车辆状态查看 | 实时查看电量、续航、充电状态、门窗状态 |
| 充电管理 | 查看充电进度，设置充电时间和充电限值 |
| 车钥匙共享 | 通过 App 授权他人使用蓝牙数字钥匙，可设置权限和有效期 |

远程控制需要车辆处于联网状态（SIM 卡流量或 Wi-Fi），车辆断网时远程指令无法送达。

## OTA 升级

极氪车型支持 OTA（空中升级）推送，无需到店即可更新系统。

- 收到 OTA 推送通知后，在极氪 App 或车机内查看更新内容。
- 可设置预约安装时间（如夜间充电时自动安装），安装过程中车辆不可使用。
- OTA 升级建议在电量高于 30%、车辆静止且网络信号良好时进行。
- 如 OTA 失败，可重试；多次失败建议预约到店检查网络模块。

## 保养预约

- App 内「服务」→「预约保养」，选择保养套餐、服务中心和时间。
- 系统会根据里程和时间自动提醒保养到期。
- 保养套餐价格可在 App 内预览，实际费用以到店确认为准。

## 常见问题

| 问题 | 处理方式 |
| --- | --- |
| 远程控制无响应 | 检查手机网络，确认车辆 SIM 卡流量正常；重启 App 后重试 |
| OTA 更新一直失败 | 确认车辆停放在网络信号良好处，电量 > 30%；仍失败则预约到店 |
| 绑定车辆失败 | 核实 VIN 码是否正确，或联系门店确认车辆已入系统 |
| App 无法登录 | 检查手机号是否为注册号，尝试重置密码；仍无法登录联系客服 |
| 车辆位置不更新 | 车辆可能处于地下停车场或信号盲区，属正常现象 |

## 客服边界

- App 账号密码问题、支付问题：引导用户通过 App 官方客服渠道处理。
- 车钥匙共享权限设置：引导用户查看 App 内帮助文档，或联系销售顾问。
- SIM 卡流量套餐到期：引导用户在 App 内续费或联系极氪客服。
