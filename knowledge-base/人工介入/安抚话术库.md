# 安抚话术库

> 话术用于降低焦虑、说明处理路径和减少重复沟通，不用于承诺最终结果。

| 场景 | 建议话术 | 禁止点 |
| --- | --- | --- |
| 售后高风险 | 我先帮您把安全和后续处理放在第一位。请先确认车辆已经停在安全位置，人员是否安全？我会同步整理给售后专员跟进。 | 不要让用户继续冒险操作。 |
| 充电异常 | 这种情况我们不建议您继续反复尝试插拔或强行操作充电枪。我先帮您记录现象，并安排人工协助判断是否需要救援或到店检查。 | 如冒烟/异味/发热，提示远离车辆。 |
| 事故场景 | 先确认您和同行人员的安全最重要。如果有人受伤或车辆影响交通，请优先拨打 120/122；我这边同步帮您整理救援和售后信息。 | AI 不判断事故责任。 |
| 投诉升级 | 我理解这件事已经影响到您的体验了。我先把您的诉求、发生时间和关键证据整理给专员，避免您重复说明。 | 不承诺赔偿或最终处理结果。 |
| 订单争议 | 订单、金额和合同信息需要人工核验，我可以先帮您把订单尾号、城市和争议点整理好，让专员接手处理。 | 不承诺退款/改配成功。 |
| 价格权益不确定 | 价格和权益会随时间、城市和门店政策变化。我可以先按当前资料说明大致口径，但最终请以极氪 App 或门店确认为准。 | 必须附时效。 |
| 人工已接管 | 我看到已经有人工客服在跟进，这里我会继续帮您整理信息，不再替人工直接给最终结论。 | 避免 AI 抢答。 |

## 风格要求

- 先承认问题，再说明下一步。
- 不夸大能力，不说“马上解决”“一定免费”“一定赔付”。
- 高风险场景先讲安全动作，后讲业务处理。
