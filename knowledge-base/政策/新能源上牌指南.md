# 新能源汽车上牌指南

> 上牌政策因城市不同差异较大。以下为通用框架，具体材料清单、办理时限和地方补贴以当地车管所和极氪门店上牌专员为准。

## 新能源绿牌说明

新能源汽车使用专属绿色号牌，与燃油车蓝牌在车牌颜色、字母前缀（如"D"、"F"等）上有明显区别。绿牌车辆在部分城市享受不限行、免摇号、免竞价等政策。

## 2026-2027 年购置税政策

| 时间段 | 政策 | 减税上限 |
| --- | --- | --- |
| 2024-2025 年 | **免征**车辆购置税 | 每辆乘用车不超过 3 万元 |
| 2026-2027 年 | **减半征收**车辆购置税 | 每辆乘用车不超过 1.5 万元 |

极氪全系纯电（001、007、007GT、009、009光辉、7X、MIX、X）和插混车型（8X、9X）均在减税目录范围内。购车时需填写《新能源汽车免（减）征车辆购置税申报表》。

## 上牌所需材料（通用版）

| 材料 | 说明 |
| --- | --- |
| 购车发票原件 | 含车价和税额 |
| 车辆合格证/一致性证书 | 交付时随车附带 |
| 身份证原件 | 个人购车；单位购车需营业执照 |
| 交强险保单 | 上牌前必须购买 |
| 新能源减税申报表 | 经销商或 4S 店协助办理 |
| 车辆购置税减免凭证 | 税务机关出具或经销商代办 |

## 上牌流程

极氪已打通公安部交管 **12123 App** 全流程服务，支持「预查验」，车管所上牌无需线下验车。

标准流程：

1. **保险**：提车前购买交强险（必须）和商业险（建议）。
2. **选号**：在 12123 App 或车管所自助机选择号码，支持自编和随机，新能源专用字母前缀由系统分配。
3. **上牌**：
   - 极氪代办：提供材料给门店上牌专员，全程代办（大多数城市支持）。
   - 自行办理：携带材料到车管所，或通过 12123 App 在线办理。
4. **临牌行驶**：提车当天获得临时号牌，有效期通常 15 天，正式牌照下来前可正常上路。

## 限牌城市特殊说明

| 城市类型 | 说明 |
| --- | --- |
| 上海 | 新能源车免费获得绿牌额度，无需参与燃油车竞拍，政策每年更新，2026 年政策有效期至 2026-12-31 |
| 北京 | 新能源摇号指标，摇中后方可购买；与燃油车指标分开，排队等待周期长 |
| 广州、深圳 | 新能源限牌但有单独指标，具体以当年政策为准 |
| 其他城市 | 大多数城市新能源车不限购不限行，可直接上牌 |

客服只提供框架说明，具体城市指标余量和申请资格以当地政务平台为准，不做承诺。

## 外地购车上牌

- 户籍地与购车城市不同时，一般在车辆使用地（居住地）上牌。
- 部分限牌城市（如北京）要求必须持有本市指标才能上牌。
- 外地用户购车前应先确认本地政策，客服可提示注意，但不提供具体城市指标信息。

## 客服边界

- 购置税减免金额计算：告知政策框架（减半、上限 1.5 万），具体金额引导用户咨询税务或门店财务。
- 限牌城市指标申请：说明需通过当地政务平台申请，不在极氪客服服务范围内。
- 上牌材料遗漏或异常：引导用户联系门店上牌专员处理。
